Funcionalidades Principais

Notificações quando quotas atingem 80%, 90% ou valores personalizados.

Configurável

Todas as configurações via settings.json do VS Code. Simples e familiar.

Painéis Reutilizáveis

Open Source

Configuração

Fácil de Configurar

settings.json 
{
  "techquotas.providers": {
    "openai": {
      "apiKey": "${OPENAI_API_KEY}",
      "warningThreshold": 80,
      "criticalThreshold": 95
    },
    "anthropic": {
      "apiKey": "${ANTHROPIC_API_KEY}"
    },
    "gemini": {
      "apiKey": "${GEMINI_API_KEY}"
    },
    "groq": {
      "apiKey": "${GROQ_API_KEY}"
    }
  },
  "techquotas.refreshInterval": 60000
}
